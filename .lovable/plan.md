

# PPT 机械布局图改进：清晰表达相机与执行机构的空间关系

## 当前问题分析

1. **相机与执行机构距离太近**：挂载相机仅偏移 80px，与机构图标重叠难以区分
2. **连接线语义单一**：只有"安装于"标签，缺少安装方式（顶装/侧装/支架）信息
3. **拍摄方向线太弱**：opacity 0.4 的虚线在深色背景上几乎不可见
4. **缺少视觉层级**：相机和机构使用相同大小的标签圆圈，无法一眼区分主次
5. **相机朝向不明**：相机图标是静态矩形，看不出镜头对准哪个方向
6. **挂载关系与拍摄关系混淆**：两种线没有足够视觉差异

## 改进方案（`src/components/canvas/SimpleLayoutDiagram.tsx`）

### 1. 相机图标增加朝向指示
给 CameraIcon 增加一个三角形"视野锥"，从镜头中心指向产品方向，用半透明蓝色扇形表示视场角（FOV），让人一眼看出相机拍的是哪里。

### 2. 安装关系线增加安装方式标注
将"安装于"改为具体安装方式（如"顶装"、"侧装"、"支架"），从 `cameraMounts` 数据中读取。连接线改用双色渐变（相机蓝→机构色），视觉上更像"物理连接"。

### 3. 拍摄方向线加强可见度
- opacity 从 0.4 提升到 0.6
- strokeWidth 从 1.5 提升到 2
- 在箭头终点处（产品侧）加一个小圆点标记拍摄落点
- 线上标注"拍摄"文字

### 4. 挂载相机位置优化
- 挂载偏移从 80px 增加到 110px，避免图标重叠
- 相机旋转朝向产品中心，视觉上更自然

### 5. 增加"安装组"视觉分组
为每对已挂载的相机+机构，画一个浅色半透明圆角矩形背景框，把它们框在一起，形成视觉上的"安装组"，一眼能看出哪些硬件是绑在一起的。

### 6. 图例增加关系说明
在右侧面板底部增加线型图例：
- 实线蓝色 = 安装关系
- 虚线青色 = 拍摄方向
让不熟悉的人也能快速理解图上的线代表什么。

## 不变项
- 整体拓扑布局逻辑、图标大小、信息面板结构不变
- PPT 生成调用方式不变
