

# 相机位于执行机构上方（最上层）

## 问题
当前挂载相机的位置是沿着"机构到中心"的径向方向向外偏移 150px，这意味着相机可能出现在机构的任意方向（左、右、下方等），而不是始终在机构的正上方。

## 改动方案

**文件**：`src/components/canvas/SimpleLayoutDiagram.tsx`

### `buildTopologicalLayout` 函数中挂载相机的定位逻辑（约第 269-275 行）

将当前的径向偏移逻辑改为：**挂载相机始终放在其执行机构的正上方**，Y 坐标减去固定偏移（如 120px），X 坐标与机构对齐。

```typescript
// 当前：沿径向方向偏移
camX = mounted.x + (dx / dist) * 150;
camY = mounted.y + (dy / dist) * 150;

// 改为：始终在机构正上方
camX = mounted.x;
camY = mounted.y - 120;
```

这样所有挂载相机都会统一出现在其对应机构的正上方，视觉上形成"相机在最上层"的层次关系。

### SVG 渲染顺序确认
当前渲染顺序已经是：安装组背景 → 连接线 → 产品 → 机构 → 相机。相机已经是最后渲染的（最上层），无需调整。
