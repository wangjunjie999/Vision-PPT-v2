

# 机械臂 3D 模型精细化升级

## 当前状态

现有 `RobotArmModel` 使用简单的圆柱+球体组合，结构为：底座 → 腰部 → 肩关节 → 大臂 → 肘关节 → 小臂 → 腕关节 → 末端段 → 法兰。整体偏简陋，缺乏工业机械臂的细节特征。

## 升级内容

### `src/components/canvas/Layout3DPreview.tsx` — `RobotArmModel` 函数

**1. 底座增强**
- 底座改为双层结构：底部宽厚底板 + 上部带散热槽的圆柱体
- 添加安装螺孔标记（4-6个小圆柱凸起在底板边缘）
- 底座侧面添加品牌标志区域（深色矩形面板）

**2. 臂段造型优化**
- 大臂/小臂从简单圆柱改为带护罩的工字型截面：两端粗中间细的锥形过渡
- 每段臂添加侧面加强筋（薄片 Box）
- 臂段表面添加通风散热孔装饰（小型矩形凹槽阵列）

**3. 关节细化**
- 肩/肘/腕关节从单个球体改为双盘夹球结构：两个薄圆盘 + 中间球体
- 关节两侧添加驱动电机外壳（小圆柱体突出于关节侧面）
- 关节处添加旋转刻度环（极薄圆环，颜色区分）

**4. 线缆管理**
- 现有单根线缆改为线缆束（2-3根不同粗细的管线）
- 在关节处添加线缆固定夹（小型环状几何体）
- 末端段添加气管/信号线分支

**5. 法兰末端精细化**
- 法兰盘添加螺栓孔标记（6-8个均匀分布的小圆柱）
- 法兰与适配器之间添加定位销标记
- 法兰顶面添加十字定位标线

**6. X-Ray 模式同步更新**
- 所有新增几何体在 xray 分支中同步添加线框版本

### 技术要点
- 仅修改 `RobotArmModel` 函数（约 140 行），不影响其他机构模型
- 所有尺寸基于 `w/h/d` 参数比例计算，保持自适应
- 使用现有 `mechMat`/`rubberMat` 材质函数，增加少量新颜色常量
- 几何体段数适度（12-20），平衡视觉效果和渲染性能
