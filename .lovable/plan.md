

# 产品渲染层级与吸附颜色

## 问题
产品目前在机构**下方**渲染（line 1978），吸附后颜色变化不够明显。应像相机一样：渲染在机构**上方**，吸附后变色+半透明。

## 改动（单文件）

**`src/components/canvas/DraggableLayoutCanvas.tsx`**

1. **调整渲染顺序**：将产品的渲染块（当前 ~line 1978-2040）移到机构渲染块之后、相机渲染块之前，使层级为：机构 → 产品 → 相机。

2. **吸附后视觉效果**（对标相机的处理方式）：
   - 吸附后 `opacity={0.7}`（与相机一致，line 2145）
   - 吸附后填充色从 `url(#product-grad)` 改为纯色 `#16a34a`（绿色，与相机吸附色一致）
   - 吸附后描边从 `#22d3ee` 改为 `#22c55e`（绿色）
   - 未吸附时保持原有青色外观

3. **产品渐变定义**：保留 `product-grad`，仅在未吸附时使用。

渲染顺序最终为：
```text
底层 ← 机构(mechanism) → 产品(product) → 相机(camera) → 顶层
```
