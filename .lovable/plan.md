
# 三视图层次化布局方案

## 现状分析

当前三视图编辑器（`DraggableLayoutCanvas`）使用 1:1 比例 + 照片级机构图片渲染，存在以下问题：

1. **层次不清晰** - 所有对象（相机、机构、产品）在同一平面混在一起，没有视觉层次区分
2. **逻辑关系不明确** - 相机与其对应的机构/产品之间的安装关系、检测关系仅靠虚线表达，不够直观
3. **数据与图形混杂** - 坐标数字直接标在图上，增加视觉噪音
4. **三张图之间缺乏联动** - 正视图、侧视图、俯视图各自独立，无法直观对应同一对象在不同视图中的位置

## 借鉴方案

参考专业工程图纸和 CAD 程序的三视图排列方式：

### 1. 国标三视图排列（第一角投影 / 第三角投影）

工程制图中三视图有标准排列方式：
- 正视图居左上
- 俯视图在正视图正下方
- 侧视图在正视图正右方
- 三图共享投影线，读者可以沿水平/垂直方向对齐同一个零件

### 2. 层级视觉编码

- **Z 轴分层**：通过颜色深浅、阴影大小区分不同高度层的对象
- **区域分区**：产品检测面用浅色高亮底色标记，机构活动区域用虚线框标记
- **连接关系**：使用标号系统（而非连线）表示对应关系，如 "A1" 在正视图标记相机，在侧视图同一相机也标 "A1"

### 3. 尺寸说明独立区域

将尺寸标注集中到图纸边缘的"尺寸表"中，图上仅保留编号引用线。

## 实施方案

### 步骤 1：新建三合一联动视图组件

**新建文件**: `src/components/canvas/ThreeViewLayout.tsx`

创建一个将三张视图合并到同一画面的组件，布局如下：

```text
+-------------------+------------------+
|                   |                  |
|    正视图 (X-Z)    |   侧视图 (Y-Z)   |
|                   |                  |
+-------------------+------------------+
|                   |                  |
|    俯视图 (X-Y)    |   尺寸说明表     |
|                   |                  |
+-------------------+------------------+
```

关键特性：
- 四宫格布局，三视图按工程制图标准排列
- 投影对齐线：正视图与俯视图共享 X 轴垂直对齐线；正视图与侧视图共享 Z 轴水平对齐线
- 每个视图使用统一比例（自动计算 fitScale 使所有对象都在视图范围内）
- 右下角为"尺寸说明表"，集中显示所有关键尺寸

### 步骤 2：层次化视觉设计

为不同类型的对象设计清晰的视觉层次：

| 层级 | 对象类型 | 视觉表现 | 编号规则 |
|------|---------|---------|---------|
| 1 (底层) | 传送带/基台 | 灰色填充，最低对比度 | M1, M2... |
| 2 | 机构（气缸/夹爪/升降台等） | 彩色描边 + 浅色填充 | M3, M4... |
| 3 | 产品 | 青色高亮 + 十字线 | P |
| 4 (顶层) | 相机 + 光源 | 蓝色实心 + 视场锥 | C1, C2... |

每一层用不同的透明度和阴影深度实现"深度感"。

### 步骤 3：标号对应系统

在三张视图中使用统一的标号系统替代直接数据标注：

- 每个对象分配一个简短标号（C1, C2, M1, M2...）
- 标号在三张图中保持一致，用圆圈内数字表示
- 右下角尺寸说明表列出每个标号对应的：
  - 名称
  - 3D 坐标 (X, Y, Z)
  - 到产品中心的距离
  - 特殊关系（如 "C1 安装在 M2 上"）

### 步骤 4：修改 SimpleLayoutDiagram 用于 PPT

**修改文件**: `src/components/canvas/SimpleLayoutDiagram.tsx`

将 `SimpleLayoutDiagram` 从单视图概览改为三合一视图：
- 左上：正视图（简化图标）
- 右上：侧视图（简化图标）
- 左下：俯视图（简化图标）
- 右下：保留现有关键参数面板
- 添加投影对齐线连接三个视图
- 对象使用统一标号系统

### 步骤 5：在 DraggableLayoutCanvas 中添加"概览模式"

**修改文件**: `src/components/canvas/DraggableLayoutCanvas.tsx`

在视图切换栏中增加一个"三合一概览"按钮：
- 点击后切换到只读的三合一视图模式
- 显示 `ThreeViewLayout` 组件
- 可以截图保存为 PPT 素材
- 再次点击或点击单视图标签返回编辑模式

### 步骤 6：更新 PPT 生成逻辑

**修改文件**: `src/services/pptx/workstationSlides.ts`

布局页使用三合一视图图片（而非单张概览图），使 PPT 读者可以从一张图理解完整的空间布局关系。

## 需要修改的文件

1. **新建** `src/components/canvas/ThreeViewLayout.tsx` - 三合一联动视图组件
2. **修改** `src/components/canvas/SimpleLayoutDiagram.tsx` - 改为三合一布局 + 标号系统
3. **修改** `src/components/canvas/DraggableLayoutCanvas.tsx` - 添加概览模式入口
4. **修改** `src/services/pptx/workstationSlides.ts` - 使用三合一视图图片

## 预期效果

- 一张图即可看清所有硬件在三维空间中的位置关系
- 投影对齐线帮助读者在三视图之间追踪同一对象
- 标号系统消除图上数据杂乱感
- 尺寸说明表集中提供精确数据
- PPT 中减少翻页次数，信息更集中
