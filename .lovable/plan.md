

# 浅色模式高级灰色调优化

## 问题分析

当前浅色模式中大量使用纯白或接近纯白的颜色：
- `--background: 220 15% 98%` — 几乎纯白
- `--card: 0 0% 100%` — 纯白
- `--popover: 0 0% 100%` — 纯白
- `--toolbar-bg: 0 0% 100%` — 纯白
- `--panel-bg: 215 15% 99%` — 几乎纯白

## 调整方案

将所有浅色模式的白色背景替换为带有蓝灰色调的高级浅灰，保持"系统制图学"设计语言的冷调工业感：

**文件：`src/index.css`** — 修改 `:root` 下的 CSS 变量：

| 变量 | 当前值 | 新值 | 说明 |
|------|--------|------|------|
| `--background` | `220 15% 98%` | `220 14% 95%` | 整体背景降至浅灰 |
| `--card` | `0 0% 100%` | `220 13% 97%` | 卡片带蓝灰调 |
| `--popover` | `0 0% 100%` | `220 13% 97%` | 弹窗同步 |
| `--secondary` | `215 15% 93%` | `215 14% 90%` | 次级元素更明显 |
| `--panel-bg` | `215 15% 99%` | `215 14% 96%` | 面板背景 |
| `--panel-header` | `215 18% 96%` | `215 16% 93%` | 面板标题区 |
| `--toolbar-bg` | `0 0% 100%` | `220 13% 97%` | 工具栏 |
| `--tree-hover` | `215 20% 95%` | `215 18% 92%` | 树节点悬停 |
| `--tree-selected` | `220 45% 94%` | `220 40% 91%` | 树节点选中 |
| `--slot-empty` | `215 15% 97%` | `215 14% 94%` | 空槽位 |
| `--slot-filled` | `220 40% 95%` | `220 35% 92%` | 已填充槽位 |
| `--slot-hover` | `220 30% 93%` | `220 28% 90%` | 槽位悬停 |
| `--input` | `215 15% 89%` | `215 14% 87%` | 输入框底色 |
| `--gradient-steel` | `97%→92%` | `94%→89%` | 钢铁渐变加深 |

所有调整统一降低亮度 3-4%，并保留蓝灰色相（220-215 hue），确保层级关系不变（背景 < 卡片/面板 < 工具栏元素），视觉上更沉稳、高级。
